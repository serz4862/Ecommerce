# MERN STACK PROJECT

## Ecommerce website

Command 1: npx create-react-app ecommerce. (for react ).<br>
Command 2: npm install @mui/material @emotion/react @emotion/styled ( for material-UI).<br>
Command 3: npm install @mui/icons-material @mui/material @emotion/styled @emotion/react (for material-UI).<br>
Command 4: npm install -D tailwindcss (for Tailwind Css).<br>
Command 5: npm install @headlessui/react.<br>
Command 6: npm install @heroicons/react.<br>
Command 7: npm i react-alice-carousel.<br>
